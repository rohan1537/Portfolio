# Portfolio Website - Rohan S

A personal portfolio website built with HTML and CSS to showcase projects, skills, and academic background.

## Project Overview
This is a single-page responsive portfolio developed as part of my B.Tech AI coursework at Amrita Vishwa Vidyapeetham. It highlights my experience in programming (Python, C, Java) and my interest in AI and hardware performance.

## Core Sections
* **Hero**: Introduction and primary contact links.
* **About**: Career objectives and academic focus.
* **Skills**: Categorized technical expertise in languages and AI tools.
* **Projects**: Summary of key software and hardware-related projects.
* **Education**: Academic history and performance metrics.
* **Interests**: Personal hobbies and technical interests.

## Technologies Used
* HTML5
* CSS3 (Flexbox & Grid)
